# Semi-Supervised-Fuzzily-Weighted-Adaptive-Boosting-for-Classification

MATLAB code for the research paper titled "Semi-Supervised Fuzzily Weighted Adaptive Boosting for Classification".
This work is described in:
===============================================================================================================
Gu, X., Angelov, P., & Shen, Q. (2024). Semi-Supervised Fuzzily Weighted Adaptive Boosting for Classification. 
IEEE Transactions on Fuzzy Systems, DOI: 10.1109/TFUZZ.2024.3349637
===============================================================================================================
Please cite the paper above if this code helps.
